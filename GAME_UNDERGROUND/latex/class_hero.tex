\doxysubsection{Detailed Description}
Class to contain the hero. 

Contains the name of the hero, its coordinates, the pointer to the table with the basic parameters, the pointer of the weapon, the number of keys, the potion number, max potion number and 2 map-\/s with equipment and potion. \textbackslash{}datails Builds with the \mbox{\hyperlink{class_table}{Table}} OR with the \mbox{\hyperlink{class_table}{Table}} + weapon OR with the \mbox{\hyperlink{class_table}{Table}} + weapon + equipment. \begin{DoxyAuthor}{Author}
Lina\+AngelinaG 
\end{DoxyAuthor}

15.\+12.\+2021 
\end{DoxyDate}

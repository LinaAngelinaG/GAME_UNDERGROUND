Detailed Description

Class that desribes all the weapon that upgrates the hero's characteristics.

Contains the map-like-class Modificate that includes parameters and values in which

these parameters change(from the artefact class). Also contains function use() from the class weapon.

Builds with the name of the weapon, the name of the parameter and its value.

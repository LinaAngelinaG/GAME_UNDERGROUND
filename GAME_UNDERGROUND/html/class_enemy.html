Detailed Description

Class that desribes enemies.

Contains the name of the enemy, its power, health, current_health, experience, the amount outcomes and the pointer of the tool tha the enemy holds.

Builds with the name of the enemy, other basic parameters are declared for each enemy, but you can change it.

Detailed Description

Class to contain the hero.

Contains the name of the hero, its coordinates, the pointer to the table with the basic parameters, the pointer of the weapon, the number of keys, the potion number, max potion number and 2 map-s with equipment and potion. \datails Builds with the Table OR with the Table + weapon OR with the Table + weapon + equipment.

Detailed Description

Class that desribes all the weapon that upgrates the hero's characteristics and that gives the special bonus against some enemies.

Contains the map-like-class Modificate that includes parameters and values in which

these parameters change(from the artefact class). And containes the table of charmes and its bonus' values.

Builds with the name of the weapon, the charm's name (optional using value of the charm) the name of the parameter and its value.

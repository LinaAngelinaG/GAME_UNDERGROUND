Detailed Description

template<typename T>
class Container< T >

Template class that holds the binary tree that describes the map template-class.

Friend-class for the class node.

Holds the root of the tree, its size and the own iterator-class.

Builds with the key and its value of the node or as an empty-container without any parameters.
